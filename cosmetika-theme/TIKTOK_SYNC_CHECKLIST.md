# TikTok Shop Sync — Pre-Launch Checklist

## 1. Shopify Sales Channel Setup
- [ ] Install "TikTok" sales channel from Shopify App Store
- [ ] Connect TikTok for Business account
- [ ] Authorize catalog sync (Products → TikTok Shop)
- [ ] Set up TikTok Pixel via the sales channel (auto-injects; manual pixel in theme.liquid is backup)

## 2. Product Requirements (every product MUST have)
- [ ] **Title** — clean, no ALL CAPS, no special characters
- [ ] **Description** — minimum 1 sentence (TikTok rejects blank descriptions)
- [ ] **Images** — at least 1, white background preferred, min 500x500px
- [ ] **Price** — set and >$0
- [ ] **Inventory** — tracked, with stock count >0
- [ ] **SKU** — unique per variant (TikTok uses this as product identifier)
- [ ] **Weight** — required for shipping calculation
- [ ] **Product type** — maps to TikTok category
- [ ] **Barcode/GTIN** — strongly recommended for discovery

## 3. Collection Structure
- [ ] Create "All Products" collection (automatic: all conditions)
- [ ] Create "Best Sellers" collection (for homepage grid)
- [ ] Create "New Arrivals" collection
- [ ] Ensure collections are set to publish to TikTok sales channel

## 4. Policies (required by TikTok Shop)
- [ ] Return Policy page created
- [ ] Shipping Policy page created
- [ ] Privacy Policy page created
- [ ] Terms of Service page created

## 5. Post-Sync Verification
- [ ] Products visible in TikTok Seller Center
- [ ] Prices match Shopify (no discrepancy)
- [ ] Images loaded correctly
- [ ] Test purchase flow from TikTok → Shopify checkout
- [ ] Verify inventory deduction after test order

## 6. Livestream Prep
- [ ] Pin featured products in TikTok LIVE shopping tab
- [ ] Set flash deal prices (if applicable) via Seller Center
- [ ] Test LIVE shopping overlay with product cards
- [ ] Announcement bar on Shopify set to "LIVE ON TIKTOK — Shop the Launch"

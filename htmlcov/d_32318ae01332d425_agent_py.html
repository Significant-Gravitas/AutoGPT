14class Agent: 

15 """Agent class for interacting with Auto-GPT. 

16 

17 Attributes: 

18 ai_name: The name of the agent. 

19 memory: The memory object to use. 

20 full_message_history: The full message history. 

21 next_action_count: The number of actions to execute. 

22 system_prompt: The system prompt is the initial prompt that defines everything 

23 the AI needs to know to achieve its task successfully. 

24 Currently, the dynamic and customizable information in the system prompt are 

25 ai_name, description and goals. 

26 

27 triggering_prompt: The last sentence the AI will see before answering. 

28 For Auto-GPT, this prompt is: 

29 Determine which next command to use, and respond using the format specified 

30 above: 

31 The triggering prompt is not part of the system prompt because between the 

32 system prompt and the triggering 

33 prompt we have contextual information that can distract the AI and make it 

34 forget that its goal is to find the next task to achieve. 

35 SYSTEM PROMPT 

36 CONTEXTUAL INFORMATION (memory, previous conversations, anything relevant) 

37 TRIGGERING PROMPT 

38 

39 The triggering prompt reminds the AI about its short term meta task 

40 (defining the next task) 

41 """
